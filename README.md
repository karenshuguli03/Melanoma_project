# Melanoma Skin Cancer Detection

This project implements a **skin cancer detection system** using deep learning. We compare a **base CNN**, **EfficientNet-B0**, and a **Transformer-based model** to classify lesions as benign or malignant.

## Dataset
Download the dataset from:

- **Google Drive**: [Shared Melanoma Dataset](https://drive.google.com/drive/folders/1kLYAv9jxFuBII54V8XChU7i9a2x-wLAC)  
  Place the `train` and `test` folders inside a folder named `melanoma`.
- **Kaggle**: [Melanoma Dataset on Kaggle](https://www.kaggle.com/datasets/hasnainjaved/melanoma-skin-cancer-dataset-of-10000-images/data)
## Models

- **Base CNN** – baseline for classification.  
- **EfficientNet-B0** – lightweight CNN with transfer learning.  
- **Transformer-based model** – explores global dependencies for classification.
